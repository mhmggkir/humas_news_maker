## Cara Penggunaan Aplikasi

1. Aplikasi memerlukan koneksi internet
2. Anda harus memiliki API Key dari [Groq](groq.com)
3. _Prompt_, _model_, _port_, dan _news structure_ dapat diubah di file config.json
